== Einführung
In diesem Dokument werden die wesentlichen Begriffe aus dem Anwendungsgebiet (Fachdomäne) der <Projektthema-Systems> definiert. Zur besseren Übersichtlichkeit sind Begriffe, Abkürzungen und Datendefinitionen gesondert aufgeführt.

== Begriffe aus dem Habit Tracker Domainmodell
[%header]
|===
| Begriff | Synonyme / Verwandte Begriffe | Definition und Erläuterung

| User
| Benutzer, Nutzer, Account-Inhaber
| Eine Person, die den Habit Tracker verwendet, um ihre Gewohnheiten zu verwalten und zu überwachen.

| Habit
| Gewohnheit, Routine, Habit (englisch), Zielhandlung
| Eine spezifische Handlung oder Verhaltensweise, die der Nutzer regelmäßig ausführen möchte.

| HabitLog
| Eintrag, Tageslog, Fortschrittsnachweis, Erfolgseintrag
| Ein Dokumentationseintrag, der die Durchführung einer Gewohnheit an einem Tag festhält, inklusive Erfolg oder Misserfolg.

| Category
| Kategorie, Themenbereich, Gruppe, Label
| Eine Gruppierung oder Klassifikation von Gewohnheiten, z.B. Gesundheit, Fitness.

| Beziehungen
| Beziehungen, Assoziationen, Verbindungen
| Beschreiben die Verknüpfungen zwischen den Entitäten:
| - Benutzer besitzt Gewohnheiten
| - Gewohnheit hat Log-Einträge
| - Gewohnheit gehört zu Kategorien
|===
