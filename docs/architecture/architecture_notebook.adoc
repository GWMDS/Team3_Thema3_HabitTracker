Das System soll als eine Progressive Web App (PWA) umgesetzt werden, um die Kompalibiltät mit möglichst vielen Geräten zu gewährleisten.

- *Plattformunabhängigkeit:* die Anwendung läuft im Webbrowser und soll auf PCs, Tablets und Smartphones bedienbar sein (-> Responsive Design)
- *Offline-Nutzung:* durch einen Service-Worker cacht die PWA alle Daten, sodass eine Nutzung ohne Internetverbindung möglich ist
- *einfache Bereitstellung:* durch den Webansatz ist keine komplizierte Installation notwendig (Nutzer können die App einfach als Icon auf den Homescreen hinzufügen)

Kritische Fragen:

- ist neben der PWA ein zusätzliches Backend nötig (Datenbank etc.) oder nur clientseitiges Arbeiten?
- gibt es sonst noch irgendwelche externen APIs auf die man zugreifen müsste?
