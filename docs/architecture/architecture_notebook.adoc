== Schichten oder Architektur-Framework
Wir verwenden eine klassische 3-Schichten-Architektur:

1. **Präsentationsschicht (Client/PWA)**  
   - Vue.js SPA  
   - Kommunikation über HTTP mit REST-API  
//- Service Worker für Offline-Funktionalität  

2. **Logikschicht (Backend/API)**  
   - Node.js + Express  
   - REST-Endpoints zum Lesen/Schreiben von Daten  

3. **Datenhaltungsschicht**  
   - JSON-Dateien im Dateisystem
   - SQLite
   - Option für spätere Migration zu PostgreSQL
